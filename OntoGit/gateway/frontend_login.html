Gateway Login

数据中台统一入口

登录入口现在收口到 gateway。登录成功后，默认进入中台总览页；同时保留对版本治理、版本图谱和概率服务的统一访问入口。

统一入口 登录态由 gateway 统一承接，再转发到后端模块。
默认落点 登录成功后默认进入 `/ui-dashboard`。
模块协同 `xiaogugit` 和 `probability` 继续保留，各自通过 gateway 统一访问。

账号登录

输入当前系统配置的账号信息后进入中台总览台。

登录成功后会保存访问令牌，并优先跳转到 `next` 参数指定页面；如果未指定，则默认进入 `/ui-dashboard`。
