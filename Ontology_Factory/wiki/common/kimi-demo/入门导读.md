# 入门导读

> 这份页面给教育页提供专题导读，也顺带把 WiKiMG 接入方案讲清楚。

## 定义与定位
入门导读页不是实体页，而是一个面向新用户的学习入口。它帮助前端把“看什么、怎么理解、为什么这样设计”整理成一个稳定结构，避免教育页完全依赖写死的 JSON 数据。

## 证据来源
- 当前 demo 先以文件真源打通浏览、图谱和问答上下文，再逐步补分析与写回。
- 系统概览、光照监测和用户画像构成了最容易理解的三步阅读路径。
